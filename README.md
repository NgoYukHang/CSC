# CSC3100: PROBLEM SOLVING AND PROGRAMMING CONCEPTS

## Using git
1. Add current directory to staging area with `git add .` 
2. Check staging area with `git status`
3. Add repo with `git remote add origin git@github.com:NgoYukHang/CSC.git`
4. Push change with `git push -u origin master`
